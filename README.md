# 🛒 FreshCart Grocery App

FreshCart is a full-stack grocery shopping web application built using the MERN stack.  
It provides a modern and responsive shopping experience where users can browse grocery items and manage cart functionality smoothly.

> 🤖 This project was developed with the help of AI-assisted tools to improve development speed, learning, debugging, and project structuring.

---

# 🚀 Features

- 🛍 Grocery product listing
- 🛒 Cart functionality
- 📱 Responsive modern UI
- ⚡ Fast Vite frontend
- 🔗 REST API integration
- 🖥 Node.js & Express backend
- 📂 Clean project structure

---

# 🛠 Tech Stack

## Frontend
- React.js
- Vite
- CSS

## Backend
- Node.js
- Express.js

## Database
- MongoDB
